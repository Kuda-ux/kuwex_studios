# KuWeX Studios - Digital Innovation Website

A modern, futuristic website built for KuWeX Studios, a digital innovation agency.

## Tech Stack

*   **Framework:** [Next.js](https://nextjs.org/) 14 (App Router)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
*   **Animations:** [Framer Motion](https://www.framer.com/motion/)
*   **Icons:** [Lucide React](https://lucide.dev/)
*   **Language:** TypeScript

## Project Structure

*   `src/app`: Main application pages (Home, Services, About, Contact) and global layout/styles.
*   `src/components`: Reusable UI components (Navbar, Footer).
*   `src/lib`: Utility functions.
*   `public`: Static assets (images, fonts).

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

To deploy to Vercel or Netlify:

1.  Push code to a Git repository.
2.  Connect the repository to your deployment provider.
3.  The build command is `npm run build` and output directory is `.next`.
