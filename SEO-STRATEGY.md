## 7. PERFORMANCE OPTIMIZATION

### Core Web Vitals Targets
- **LCP** (Largest Contentful Paint): < 2.5s
- **FID** (First Input Delay): < 100ms
- **CLS** (Cumulative Layout Shift): < 0.1

### Implementation
1. **Image optimization**: Use Next.js Image component with WebP/AVIF, lazy loading
2. **Font optimization**: Use `next/font` for Inter, font-display: swap
3. **Code splitting**: Dynamic imports for heavy components (Framer Motion)
4. **Caching**: Static generation where possible, ISR for blog
5. **CDN**: Leverage Vercel/Netlify edge network
6. **Compression**: Enable Brotli/Gzip compression
7. **Reduce JS bundle**: Tree-shake unused Lucide icons

### Mobile-First (Critical for Zimbabwe)
- 70%+ of Zimbabwe traffic is mobile
- Touch-friendly buttons (minimum 44x44px)
- Responsive images with srcset
- Optimized for slower 3G/4G connections
- Data-saving considerations (compressed images, lazy loading)
